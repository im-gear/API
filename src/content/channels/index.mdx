# Channels

Channels are the communication bridges between your agents and your audience. Makinari supports multiple channels to ensure you can reach your customers where they are.

## What is a channel?

A channel is a configured connection between Makinari and a communication platform. Once a channel is set up, agents can send and receive messages through it autonomously, or you can use the REST API and MCP tools to drive interactions programmatically.

## Inbound vs outbound

Most channels support both directions:

- **Inbound**: Messages arriving from customers are routed to the appropriate agent based on your inbound configuration.
- **Outbound**: Agents initiate contact based on AI Goals, campaign triggers, or direct API calls.

Outbound messaging on WhatsApp requires pre-approved message templates. Make sure to configure these before running outbound campaigns.

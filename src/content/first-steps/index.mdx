# First Steps

Everything you need to go from zero to a running Makinari project. Follow these steps in order the first time.

## Setup sequence

Getting started involves a few distinct phases: creating your account, configuring your project, connecting channels, and defining what you want your agents to achieve.

1. [Account Creation](/first-steps/account) - Sign up and get access to the Makinari dashboard.
2. [Project Configuration](/first-steps/project-config) - Set up your first project with brand context and default behaviors.
3. [API Keys](/first-steps/api-keys) - Generate keys for REST API and MCP Server access.
4. [Team Management](/first-steps/team) - Invite collaborators with role-based access.
5. [Outbound Configuration](/first-steps/outbound) - Configure channels your agents will use to reach out.
6. [Inbound Configuration](/first-steps/inbound) - Define how incoming messages are routed to agents.
7. [AI Goals](/first-steps/ai-goals) - Define high-level objectives and assign token budgets to agents.

You do not need to complete every step before testing. You can create an account, generate an API key, and start exploring the REST API or MCP Server right away.

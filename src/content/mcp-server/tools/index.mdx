# MCP Tools

The MCP server exposes a wide range of tools to AI models, allowing them to interact with your site's data and functionality.

## Available Tools

The following tools are available through the MCP server:

- [`generateImage`](/mcp-server/tools/generateImage): Generate images using AI.
- [`generateVideo`](/mcp-server/tools/generateVideo): Generate videos using AI.
- [`renameInstance`](/mcp-server/tools/renameInstance): Rename a robot instance.
- [`updateSiteSettings`](/mcp-server/tools/updateSiteSettings): Update site configuration.
- [`webSearch`](/mcp-server/tools/webSearch): Perform web searches.
- [`memories`](/mcp-server/tools/memories): Manage memories and context.
- [`tasks`](/mcp-server/tools/tasks): Manage tasks and to-do lists.
- [`createProject`](/mcp-server/tools/createProject): Create a new project (site).
- [`requirements`](/mcp-server/tools/requirements): Manage project requirements.
- [`leads`](/mcp-server/tools/leads): Manage leads and contacts.
- [`content`](/mcp-server/tools/content): Manage content and articles.
- [`sendEmail`](/mcp-server/tools/sendEmail): Send emails.
- [`configureEmail`](/mcp-server/tools/configureEmail): Configure email settings.
- [`configureWhatsApp`](/mcp-server/tools/configureWhatsApp): Configure WhatsApp settings.
- [`salesOrder`](/mcp-server/tools/salesOrder): Manage sales orders.
- [`sales`](/mcp-server/tools/sales): Sales-related tools.
- [`scheduling`](/mcp-server/tools/scheduling): Schedule appointments and events.
- [`getFinderCategoryIds`](/mcp-server/tools/getFinderCategoryIds): Get IDs for finder categories.
- [`analyzeICPTotalCount`](/mcp-server/tools/analyzeICPTotalCount): Analyze total count for ICP.
- [`createIcpMining`](/mcp-server/tools/createIcpMining): Create ICP mining tasks.
- [`searchRegionVenues`](/mcp-server/tools/searchRegionVenues): Search for venues in a region.
- [`webhooks`](/mcp-server/tools/webhooks): Manage webhooks.
- [`urlToMarkdown`](/mcp-server/tools/urlToMarkdown): Convert URL content to Markdown.
- [`urlToSitemap`](/mcp-server/tools/urlToSitemap): Generate sitemap from URL.
- [`segments`](/mcp-server/tools/segments): Manage customer segments.
- [`campaigns`](/mcp-server/tools/campaigns): Manage marketing campaigns.
- [`assets`](/mcp-server/tools/assets): Manage digital assets.
- [`instancePlan`](/mcp-server/tools/instancePlan): Manage instance plans.
- [`workflows`](/mcp-server/tools/workflows): Manage workflows.
- [`copywriting`](/mcp-server/tools/copywriting): Copywriting tools.
- [`sendWhatsApp`](/mcp-server/tools/sendWhatsApp): Send WhatsApp messages.
- [`whatsappTemplate`](/mcp-server/tools/whatsappTemplate): Manage WhatsApp templates.
- [`conversations`](/mcp-server/tools/conversations): Manage conversations.
- [`messages`](/mcp-server/tools/messages): Manage messages.
- [`report`](/mcp-server/tools/report): Generate reports.

# MCP Server Setup

The Model Context Protocol (MCP) server allows you to expose your tools and resources to AI models.

You have two options for setting up the MCP server:

1. **[Rest MCP compatible endpoints](/mcp-server/setup/remote)**: Connect to the hosted instance managed by Makinari.
2. **[Install Locally](/mcp-server/setup/local)**: Run the server on your own machine for development or custom deployments.

Choose the option that best fits your needs.

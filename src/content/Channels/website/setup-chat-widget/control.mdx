# Control Functions

## toggleChat()

Opens or closes the chat widget based on its current state.

**Syntax**

```javascript
window.MarketFit.toggleChat()
```

**Example**

```javascript
// Button to open/close chat
document.getElementById('chat-toggle').addEventListener('click', () => {
  window.MarketFit.toggleChat();
});
```

## sendChatMessage()

Programmatically sends a message to the chat.

**Syntax**

```javascript
window.MarketFit.sendChatMessage(content)
```

**Parameters**

- `content` string (required): Content of the message to send

**Example**

```javascript
// Send automatic message
window.MarketFit.sendChatMessage("User needs urgent assistance");
```

## setChatWelcomeMessage()

Changes the default welcome message of the chat.

**Syntax**

```javascript
window.MarketFit.setChatWelcomeMessage(message)
```

**Parameters**

- `message` string (required): New default welcome message

**Example**

```javascript
// Change message based on season
window.MarketFit.setChatWelcomeMessage(
  "Black Friday Special Offer! How can we help you?"
);
```

## hideWidget()

Hides the chat widget from the interface without removing it completely.

**Syntax**

```javascript
window.MarketFit.hideWidget()
```

**Example**

```javascript
// Hide widget temporarily
window.MarketFit.hideWidget();
```

## showWidget()

Shows the chat widget if it was previously hidden.

**Syntax**

```javascript
window.MarketFit.showWidget()
```

**Example**

```javascript
// Show widget again
window.MarketFit.showWidget();
```

## reset()

Completely resets the widget, clearing all conversations and configurations.

**Syntax**

```javascript
window.MarketFit.reset()
```

**Example**

```javascript
// Reset widget after logout
function onUserLogout() {
  window.MarketFit.reset();
}
```

# Social Networks

Integrate your social media accounts to manage communications from a central platform.

## Social Platforms

- **[Facebook](/channels/social/facebook)**: Integrate your Facebook page.

## Supported Platforms

- Facebook
- Twitter
- LinkedIn

## Configuration

To configure social media channels, you need to connect your accounts in the site settings.

### Site Settings

Update your site settings to include your social media profiles:

```json
{
  "channels": {
    "social": {
      "facebook": "https://facebook.com/yourpage",
      "twitter": "https://twitter.com/yourhandle",
      "linkedin": "https://linkedin.com/company/yourcompany"
    }
  }
}
```

### Features

- **Unified Inbox**: View messages from all social channels in one place.
- **Automated Responses**: Set up automated replies for common inquiries.

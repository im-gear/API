# Webflow Integration

Connect your Webflow site to allow agents to read content, update CMS items, and manage site settings.

## Prerequisites

1.  **Webflow Account**: You must have a Webflow account with at least one active site.
2.  **Site Admin Access**: You need administrative privileges to generate API tokens.

## Configuration

### Step 1: Generate API Token

1.  Log in to your [Webflow Dashboard](https://webflow.com/dashboard).
2.  Open the **Settings** for the site you want to connect.
3.  Go to the **Integrations** tab.
4.  Scroll down to the **API Access** section.
5.  Click **Generate API Token**.
6.  Copy the token immediately (you won't be able to see it again).

### Step 2: Connect to Uncodie

Provide the API Token to the configuration endpoint to authorize the agent.

> **Note**: Ensure you treat your API Token like a password. Do not share it publicly.

## Features

Once connected, the agent can:
-   **Read CMS Collections**: Analyze your site's content to answer user queries.
-   **Update Content**: Modify CMS items (if permitted by agent role).
-   **Sync Forms**: Process form submissions from your Webflow site.

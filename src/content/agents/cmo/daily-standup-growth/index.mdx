# Daily Standup — Growth Analysis

Analyzes growth metrics, content performance, experiments, and campaigns to provide a daily strategic report.

`POST /api/agents/cmo/dailyStandUp/growth`

**Request Body**

```json
{
  "site_id": "123e4567-e89b-12d3-a456-426614174000"
}
```

| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `site_id` | `string` | Yes | The UUID of the site to analyze. |

**Response**

Returns an analysis of content creation, active experiments, running campaigns, and growth strategy alignment.

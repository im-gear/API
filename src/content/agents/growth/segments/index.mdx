# Segments

Analyzes and identifies profitable audience segments for the website.

`POST /api/agents/growth/segments`

**Request Body**

```json
{
  "siteId": "123e4567-e89b-12d3-a456-426614174000",
  "segmentData": {
    "segmentCount": 5
  }
}
```

| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `siteId` | `string` | Yes | The UUID of the site. |
| `userId` | `string` | No | The UUID of the user. |
| `agent_id` | `string` | No | The UUID of the specific agent. |
| `segmentData` | `object` | No | Configuration for segmentation (e.g., `segmentCount`). |

**Response**

Returns a list of identified and created audience segments.

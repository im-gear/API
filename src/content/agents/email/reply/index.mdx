# General Reply

Handles replies for the main agent address.

`POST /api/agents/email/reply`

**Request Body**

```json
{
  "site_id": "123e4567-e89b-12d3-a456-426614174000",
  "limit": 10
}
```

| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `site_id` | `string` | Yes | The UUID of the site. |
| `limit` | `number` | No | Max emails to process (default: 10). |

**Response**

Returns details of the processed replies and emails sent.

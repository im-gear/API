# Robots API

The Robots API manages Scrapybara robot instances, allowing you to create, control, and monitor automated browser sessions.

## Overview

The Robots API provides endpoints for:

- **Instance Management**: Create, stop, pause, resume, and delete robot instances.
- **Plan Management**: Initiate, execute, and verify automation plans.
- **Authentication**: Manage browser authentication sessions.
- **Utilities**: Helper endpoints like stream URL retrieval.

# Visitors API

The Visitors API allows you to track, identify, and segment visitors on your site, enabling personalized experiences and analytics.

## Overview

The Visitors API includes endpoints for:

- **Tracking & Identification**: Track events, identify visitors, and manage segments.
- **Session Management**: Create, update, and manage visitor sessions.
- **Sites & Uploads**: Manage sites and upload visitor-related files.
- **Setup**: Initialize database tables and constraints.

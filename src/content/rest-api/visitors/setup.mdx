# Setup & Configuration

## Endpoints

The API includes utility endpoints for initializing the necessary database tables and constraints:

- `GET /api/visitors/setup`: Creates sites, visitors, sessions, and events tables.
- `GET /api/visitors/identify/setup`: Creates leads table and adds identification columns.
- `GET /api/visitors/session/setup`: Creates visitor_sessions table.
- `GET /api/visitors/identify/setup-constraints`: Adds unique constraints for data integrity.

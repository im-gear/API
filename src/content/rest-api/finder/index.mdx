# Finder API

The Finder API provides tools for searching and retrieving contact and company information, including role-based searches, contact lookups, and autocomplete suggestions.

## Overview

The Finder API is designed to help you discover and enrich contact data. It includes endpoints for:

- **Role Search**: Find people based on their professional roles.
- **Contact Lookup**: Retrieve email addresses and phone numbers.
- **ICP (Ideal Customer Profile)**: Analyze and retrieve ICP data.
- **Autocomplete**: Get suggestions for industries, locations, and more.

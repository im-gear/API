# Contact Lookup

## Endpoints

### Find Work Emails
`POST /api/finder/person_contacts_lookup/work_emails`

Looks up work email addresses for a specific person.

**Request Body:**
- `person_id` (string, optional): The ID of the person.
- `linkedin_public_identifier` (string, optional): The LinkedIn public identifier.
- `do_contacts_enrichment` (boolean, optional): Whether to perform contact enrichment (defaults to false).

*Note: Requires either `person_id` or `linkedin_public_identifier`.*

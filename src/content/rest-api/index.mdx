# REST API

The Makinari REST API allows you to programmatically interact with the platform's core features.

> **Note:** To use the API, you'll need an API Key. Learn how to generate one in the [First Steps: API Keys](/first-steps/api-keys) guide.

## API Sections

- **[Visitors](/rest-api/visitors)**: Track and identify site visitors.
- **[Finder](/rest-api/finder)**: Search for contacts and companies.
- **[Robots](/rest-api/robots)**: Manage and control your robot instances.
